# Air Quality Navigator

## Project Description
This project predicts Air Quality Index (AQI) using machine learning and real-time API data.

## Features
- Fetch real-time air pollution data (OpenWeather API)
- Predict AQI using Linear Regression
- Backup using dataset if API fails
- User-friendly Streamlit interface

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- OpenWeather API

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Dataset
- city_day.csv (air pollution dataset)

## API
- OpenWeather API used for real-time data
